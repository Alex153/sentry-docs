```xml {filename:AndroidManifest.xml}
<application>
  <meta-data android:name="io.sentry.release" android:value="io.example@1.1.0" />
```

```java {tabTitle: Java}
SentryAndroid.init(this, options -> {

});
```

```kotlin {tabTitle: Kotlin}
Sentry.init {
  it.release = "io.example@1.1.0"
}
```
