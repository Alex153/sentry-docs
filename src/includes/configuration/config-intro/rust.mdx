Options are passed to the `init()` function as tuple where the first argument is the DSN and the second the options:

```rust
let _guard = sentry::init(("___PUBLIC_DSN___", sentry::ClientOptions {
    release: Some("my-project-name@2.3.12".into()),
    max_breadcrumbs: 50,
    debug: true,
    ..Default::default()
}));
```
